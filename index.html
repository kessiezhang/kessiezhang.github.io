Projects

Developed end-to-end Machine Learning projects to solve real-world business problems.

NBA Players Salaries Prediction

The objective was to build a model that could help team managers use performance metrics to better allocate salaries among players. As for players, this model can help them understand which areas they need to improve on in order to increase their salaries.

Airbnb First Booking Prediction

Ever wonder why you have certain promotion while your friends don't receive any? This business strategy is called personalized marketing. The goal of personalized marketing is to attract prospective customers and retain loyal customers. This project will attempt to classify whether or not new users will book room services within a month after they create their airbnb accounts.

Recommender System for Perfume

The goal of this project is to use natural language processing techniques to make perfume recommendations base on user's favorite perfume. Ultimately, we can not only help the users to discover cheaper alternatives to their favorite perfume, but also help them discover their signature perfumes.

Hotel Reviews Analysis

A lot of users have a difficult time finalizing their decisions when it comes to booking hotels, especially if there are many highly rated hotels for them to choose from. In order to resolve this problem, I built a model to extract the pros and cons words of each hotel to help end users make better and faster decisions.
